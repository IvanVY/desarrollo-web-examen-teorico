50 términos fundamentales para desarrolladores

Autor: Ivan Vera

Conceptos clave del desarrollo web

Este glosario recoge los 50 términos más importantes que todo desarrollador web debe conocer, desde fundamentos básicos hasta conceptos avanzados.

# Término Definición
1HTMLLenguaje de marcado estándar para crear páginas web. Define la estructura y el contenido mediante etiquetas.
2CSSLenguaje de hojas de estilo que describe cómo se presentan los elementos HTML en pantalla.
3JavaScriptLenguaje de programación interpretado que permite crear contenido dinámico e interactivo en páginas web.
4DOMModelo de Objetos del Documento, representación jerárquica en memoria de la estructura de un documento HTML.
5APIInterfaz de Programación de Aplicaciones que permite la comunicación entre componentes de software.
6ServidorComputadora o sistema que provee servicios, recursos o datos a otras computadoras (clientes) en una red.
7ClienteDispositivo o software que solicita servicios o recursos a un servidor, como un navegador web.
8GETMétodo HTTP para solicitar datos de un recurso específico. Los parámetros van en la URL.
9POSTMétodo HTTP para enviar datos al servidor para crear/actualizar recursos. Los datos van en el cuerpo.
10issetFunción en PHP que determina si una variable está declarada y no es NULL.
11PHPLenguaje de scripting del lado del servidor especialmente adecuado para desarrollo web.
12MySQLSistema de gestión de bases de datos relacional de código abierto muy usado con PHP.
13SQLLenguaje de consulta estructurado para gestionar y manipular bases de datos relacionales.
14HTTPProtocolo de transferencia de hipertexto, base de la comunicación de datos en la web.
15HTTPSVersión segura de HTTP que usa cifrado SSL/TLS para proteger la comunicación.
16URLLocalizador Uniforme de Recursos, dirección que identifica recursos en la web.
17DominioNombre único que identifica un sitio web en Internet (ej: ejemplo.com).
18HostingServicio que provee espacio en servidores para almacenar archivos de sitios web.
19FTPProtocolo de Transferencia de Archivos para intercambiar archivos entre cliente y servidor.
20FrontendParte cliente de una aplicación web que interactúa directamente con el usuario.
21BackendParte del servidor que procesa lógica, accede a bases de datos y genera respuestas.
22FrameworkConjunto estandarizado de conceptos y herramientas para desarrollar software.
23ReactBiblioteca JavaScript para construir interfaces de usuario basadas en componentes.
24AngularFramework de desarrollo para aplicaciones web de una sola página (SPA).
25Vue.jsFramework progresivo de JavaScript para construir interfaces de usuario.
26JSONFormato ligero de intercambio de datos basado en texto y fácil de parsear.
27XMLLenguaje de marcado extensible para almacenar y transportar datos estructurados.
28AJAXTécnica para realizar peticiones asíncronas y actualizar páginas sin recargar.
29BootstrapFramework CSS para desarrollar sitios responsive con diseño mobile-first.
30ResponsiveEnfoque de diseño que adapta la interfaz al tamaño y orientación del dispositivo.
31SEOOptimización para motores de búsqueda, técnicas para mejorar el ranking en resultados.
32Meta TagEtiquetas HTML que proporcionan metadatos sobre la página a navegadores y motores.
33CookiesPequeños archivos que los sitios web guardan en el navegador del usuario.
34SessionMecanismo para almacenar información del usuario durante su visita a un sitio.
35TokenCadena cifrada usada para autenticar y autorizar usuarios en sistemas web.
36GitSistema de control de versiones distribuido para gestionar código fuente.
37GitHubPlataforma de desarrollo colaborativo basada en Git para alojar proyectos.
38CommitOperación en Git que guarda cambios en el repositorio con un mensaje descriptivo.
39BranchRama de desarrollo independiente en un repositorio Git.
40MergeProceso de combinar cambios de diferentes ramas en Git.
41API RESTAPI que sigue principios REST, usando HTTP y formatos como JSON.
42JSON Web TokenEstándar abierto para crear tokens de acceso que permiten autenticación.
43SSLProtocolo de seguridad que establece conexiones cifradas en Internet.
44CMSSistema de Gestión de Contenidos para crear y administrar contenido digital.
45WordPressCMS popular basado en PHP y MySQL para crear sitios web y blogs.
46LaravelFramework PHP elegante y expresivo para desarrollo web con MVC.
47npmGestor de paquetes para Node.js y repositorio de librerías JavaScript.
48WebpackEmpaquetador de módulos para aplicaciones JavaScript modernas.
49TypeScriptSuperset tipado de JavaScript que compila a JavaScript puro.
50404Código de estado HTTP que indica que el recurso solicitado no existe.
